# pokemon-react
Pokedesk from pokeapi built on ReactJS

## Technologies
* React 17.0.02    
* Prop-types 15.8.1  
* React-router-dom 6.2.2  
* Eslint 8.11.0
* Jest 5.16.2
* Axios 0.26.1

## Setup
To run this project clone the repository and execute the follow commands:
```bash
npm install
cd pokedesk
npm start
```
